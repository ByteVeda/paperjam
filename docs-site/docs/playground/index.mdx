# Interactive Playground

Try paperjam directly in your browser -- no installation needed! All processing happens client-side using WebAssembly.

Upload a PDF and explore these tools:

- **[Text Extraction](text-extraction)** -- Extract text, lines, and search
- **[Table Extraction](table-extraction)** -- Detect and extract structured tables
- **[Markdown Conversion](markdown)** -- Convert PDF to Markdown
- **[Document Info](document-info)** -- View metadata and structure
- **[Split & Merge](split-merge)** -- Split PDFs by page ranges or merge multiple files
- **[Search](search)** -- Search text across all pages with highlighting
- **[Security](security)** -- Sanitize, redact, and encrypt PDFs
- **[Structure & Layout](structure-layout)** -- Analyze document structure and page layout
